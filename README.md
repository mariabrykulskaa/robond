# Robond

## Клонирование репозитория

Репозиторий содержит подмодули, поэтому при клонировании обязательно используйте флаг `--recursive`:

```bash
git clone --recursive git@github.com:robond-fintech/robond.git
```

## Генерация документации

Для генерации и просмотра документации по проекту выполните:

```bash
cargo doc --open --no-deps
```
